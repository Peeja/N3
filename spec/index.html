We introduce several concepts which extend the syntax of RDF to N3:

The set of N3 symbols consists of the set of triple elements and the set of quantifiers.

The set of N3 triple elements consists of the disjoint sets of IRIs, blank nodes, literals, universal variables, lists, quoted graphs and of one implication sign.
